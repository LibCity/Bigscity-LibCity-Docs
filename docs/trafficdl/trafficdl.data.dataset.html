class trafficdl.data.dataset.TrafficStateCPTDataset(config)¶

Bases: trafficdl.data.dataset.traffic_state_datatset.TrafficStateDataset

交通状态预测数据集的另一个基类 部分交通预测模型通过对接近度(closeness)/周期(period)/趋势(trend)进行建模实现预测。 默认使用`len_closeness`/len_period/`len_trend`的数据预测当前时刻的数据，即一个X，一个y。（一般是单步预测） 数据原始的时间戳不能为空！ 一般对外部数据进行单独建模，因此数据为[X, y, X_ext(可选), y_ext(可选)]。 默认使用`train_rate`和`eval_rate`在样本数量(num_samples)维度上直接切分训练集、测试集、验证集。

get_data()¶

获取数据，数据归一化，之后返回训练集、测试集、验证集对应的DataLoader :return:

train_dataloader (pytorch.DataLoader) eval_dataloader (pytorch.DataLoader) test_dataloader (pytorch.DataLoader) all the dataloaders are composed of Batch (class)

get_data_feature()¶

由于此类的数据输入包含`len_closeness`/len_period/`len_trend`的数据，但都融合到`X`中， 因此，继承此类的子类此函数应该返回这三段数据的具体长度（不一定等于上述的三个参数的值） :return: data_feature (dict)

class trafficdl.data.dataset.TrafficStateDataset(config)¶

Bases: trafficdl.data.dataset.abstract_dataset.AbstractDataset

交通状态预测数据集的基类 默认使用`input_window`的数据预测`output_window`对应的数据，即一个X，一个y。 一般将外部数据融合到X中共同进行预测，因此数据为[X, y]。 默认使用`train_rate`和`eval_rate`在样本数量(num_samples)维度上直接切分训练集、测试集、验证集。

get_data()¶

获取数据，数据归一化，之后返回训练集、测试集、验证集对应的DataLoader :return:

train_dataloader (pytorch.DataLoader) eval_dataloader (pytorch.DataLoader) test_dataloader (pytorch.DataLoader) all the dataloaders are composed of Batch (class)

get_data_feature()¶

返回数据集特征，子类必须实现这个函数，返回必要的特征 :return: data_feature (dict)
