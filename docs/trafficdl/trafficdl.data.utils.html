trafficdl.data.utils.py¶

trafficdl.data.utils.generate_dataloader(train_data, eval_data, test_data, feature_name, batch_size, num_workers, pad_item=None, pad_max_len=None, shuffle=True, pad_with_last_sample=False)¶
Args:

train_data, eval_data, test_data (list of input): data 中每个元素是模型单次的输入，input 是一个 list，里面存放单次输入和 target feature_name (list): 描述上面 input 每个元素对应的特征名, 应保证 len(feature_name) = len(input) batch_size (int) num_workers (int) pad_item (dict): 用于将不定长的特征补齐到一样的长度，每个特征名作为 key，若某特征名不在该 dict 内则不进行补齐。 pad_max_len (dict): 用于截取不定长的特征，对于过长的特征进行剪切 shuffle (bool) pad_with_last_sample (bool): 对于若最后一个 batch 不满足 batch_size 的情况，是否进行补齐（使用最后一个元素反复填充补齐）。
